Einlesen von Daten aus Tabellen in Markdown Format und Ausgabe in HTML

Der Sinn dieses Projekts ist die gemeinsame Entwicklung einer Software mit Hilfe der GitHub Plattform. Die Nutzung der Werkzeuge von GitHub, die Arbeit an gemeinsamen Repositories und das Teamwork selbst sind der Hauptzweck des Projekts ("Der Weg ist das Ziel").
Als Beispielprojekt soll die Entwicklung einer HTML Seite dienen, auf der mittels Javascript eine Datei im Markdown Format eingelesen wird und in ihr enthaltene Tabellen auf der HTML Seite wiedergegeben werden können. Weitere Entwicklungsschritte wären dann Manipulationen der eingelesenen Tabellen (Sortieren, Filtern etc.) sowie Interaktion mit einer Datenbank.

Zum Anzeigen der Namensliste bitte den Button drücken.
